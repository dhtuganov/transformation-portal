## M — Meetings (Деловые встречи)

Деловые встречи — самый частый формат в корпоративном мире. От небольших переговоров до масштабных тренингов.

### Типы деловых встреч

| Тип | Участники | Длительность | Особенности |
|-----|-----------|--------------|-------------|
| **Board Meeting** | 5-20 | 1-2 дня | Закрытость, VIP-сервис |
| **Management Retreat** | 20-50 | 2-4 дня | Off-site локация |
| **Training Session** | 10-100 | 1-5 дней | Оборудование, материалы |
| **Workshop** | 15-40 | 0.5-2 дня | Интерактивный формат |
| **Product Launch** | 50-500 | 1 день | Медиа, презентации |

## I — Incentives (Инсентив-программы)

<Callout type="tip" title="Что такое Incentive Travel?">
Мотивационные поездки, которые компании используют для награждения лучших сотрудников, дилеров или партнёров. Цель — создать незабываемый опыт, который невозможно купить за деньги.
</Callout>

## Практические упражнения

<Exercise number={1} title="Классификация MICE">
**Задание:** Определите сегмент MICE для каждого мероприятия:

1. Вечеринка для топ-продавцов компании на Мальдивах
2. Ежегодный съезд стоматологов
3. Заседание совета директоров в горном отеле
4. Выставка недвижимости

<Solution>
1. **Мальдивы для топ-продавцов** = **I (Incentives)** — награждение лучших, создание WOW-эффекта
2. **Съезд стоматологов** = **C (Conferences)** — профессиональное мероприятие ассоциации
3. **Совет директоров** = **M (Meetings)** — закрытая корпоративная встреча
4. **Выставка недвижимости** = **E (Exhibitions)** — экспозиция с участием застройщиков
</Solution>
</Exercise>

<Exercise number={2} title="Расчёт бюджета">
**Задание:** Клиент запрашивает организацию meeting для 30 человек на 2 дня в Алматы. Рассчитайте примерный бюджет.

**Данные:**
- Авиа: $250/чел RT
- Отель: $120/ночь
- Конф-зал: $800/день
- Кофе-брейки: $15/чел/день
- Бизнес-ланч: $25/чел/день

<Solution>
| Статья | Расчёт | Сумма |
|--------|--------|-------|
| Авиабилеты | 30 x $250 | $7,500 |
| Отель | 30 x 2 x $120 | $7,200 |
| Конф-зал | 2 x $800 | $1,600 |
| Кофе-брейки | 30 x 2 x $15 | $900 |
| Бизнес-ланч | 30 x 2 x $25 | $1,500 |
| **Subtotal** | | **$18,700** |
| Contingency 10% | | $1,870 |
| **ИТОГО** | | **$20,570** |

**На человека:** $20,570 / 30 = ~$686
</Solution>
</Exercise>

<Exercise number={3} title="Определение потребностей">
**Задание:** Клиент говорит: "Нам нужно провести мероприятие для партнёров". Какие уточняющие вопросы вы зададите?

<Solution>
**Ключевые вопросы:**

1. **Цель мероприятия?** (обучение, награждение, нетворкинг)
2. **Количество участников?**
3. **Предполагаемые даты и длительность?**
4. **Откуда приедут участники?** (1 город или разные)
5. **Нужно ли размещение?**
6. **Какой формат?** (пленарное заседание, круглые столы, воркшопы)
7. **Нужна ли развлекательная программа?**
8. **Есть ли ограничения по бюджету?**
9. **Предпочтения по локации?** (центр, загород, конкретный отель)
10. **Есть ли VIP-гости с особыми требованиями?**
</Solution>
</Exercise>
