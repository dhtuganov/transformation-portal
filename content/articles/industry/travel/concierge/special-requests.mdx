## Введение

Специальные запросы пассажиров — важная часть профессионального сервиса в туризме. Правильная обработка диетических, медицинских, религиозных требований и запросов по доступности демонстрирует экспертизу агентства и создаёт лояльных клиентов.

Для сотрудников Otrar Travel умение работать со специальными запросами критично: ошибки в этой области могут привести к серьёзным последствиям для здоровья клиента или создать неприятную ситуацию, несовместимую с его убеждениями.

## Ключевые понятия

### Dietary

Диетические требования — специальные пожелания к питанию на борту самолёта и в отеле.

**Стандартные коды питания IATA:**

| Код | Название | Описание |
|-----|----------|----------|
| AVML | Asian Vegetarian | Индийская вегетарианская кухня |
| BBML | Baby Meal | Детское питание (до 2 лет) |
| BLML | Bland Meal | Щадящее питание |
| CHML | Child Meal | Детское меню (2-12 лет) |
| DBML | Diabetic Meal | Для диабетиков |
| FPML | Fruit Platter | Фруктовая тарелка |
| GFML | Gluten-Free | Безглютеновое |
| HNML | Hindu Meal | Индуистское (без говядины) |
| KSML | Kosher Meal | Кошерное |
| LCML | Low Calorie | Низкокалорийное |
| LFML | Low Fat | С низким содержанием жира |
| LSML | Low Sodium | С низким содержанием соли |
| MOML | Muslim Meal | Халяльное |
| NLML | Non-Lactose | Без лактозы |
| VGML | Vegan | Строго вегетарианское |
| VLML | Vegetarian Lacto-Ovo | Вегетарианское с молочным |

**Правила заказа специального питания:**

1. Запрос минимум за 24-48 часов до вылета
2. Указать в PNR через SSR элемент
3. Подтвердить при регистрации
4. Зафиксировать в профиле клиента для будущих поездок

**Команда для GDS:**

```
SSR MOML KC HK1/P1
(Muslim Meal для пассажира 1 на рейсах Air Astana)
```

**Особенности по направлениям:**

- **Ближний Восток**: Халяль по умолчанию, но VLML может быть недоступно
- **Индия**: Широкий выбор вегетарианских опций
- **Европа/США**: Все типы обычно доступны, заказ за 24ч
- **Азия**: KSML может быть ограничен

### Accessibility

Доступность — обеспечение комфортного путешествия для пассажиров с ограниченными возможностями.

**Категории специальных услуг:**

| Код | Описание | Что включает |
|-----|----------|--------------|
| WCHR | Wheelchair Ramp | Коляска до трапа/гейта |
| WCHS | Wheelchair Steps | Коляска + помощь по лестнице |
| WCHC | Wheelchair Cabin | Полное сопровождение |
| BLND | Blind Passenger | Незрячий пассажир |
| DEAF | Deaf Passenger | Глухой пассажир |
| DPNA | Disabled Need Assistance | Интеллектуальные ограничения |
| MAAS | Meet and Assist | Встреча и сопровождение |

**Процедура оформления:**

1. **При бронировании**
   - Уточнить характер ограничения
   - Определить необходимый уровень помощи
   - Проверить политику авиакомпании

2. **SSR запрос**
   ```
   SSR WCHR KC HK1/P1
   (Wheelchair до гейта для пассажира 1)
   ```

3. **Дополнительная информация**
   - Вес и размеры личной коляски (если берётся с собой)
   - Тип батареи (для электроколясок)
   - Необходимость кислородного оборудования

4. **Координация с аэропортом**
   - Уведомить за 48 часов
   - Запросить подтверждение услуги
   - Рекомендовать прибытие за 3 часа

**Отели и accessibility:**

| Требование | Что проверить |
|------------|---------------|
| Мобильность | Номер на первом этаже, поручни, roll-in душ |
| Зрение | Брайль на указателях, звуковые сигналы лифта |
| Слух | Визуальные оповещения, текстовый телефон |
| Аллергия | Гипоаллергенное постельное бельё, без ковров |

### Religious

Религиозные требования — учёт вероисповедания клиента при организации путешествия.

**Основные религиозные требования:**

| Религия | Питание | Другие требования |
|---------|---------|-------------------|
| Ислам | Халяль, без алкоголя | Направление на Мекку, время молитв |
| Иудаизм | Кошерное | Шаббат (пятница вечер — суббота вечер) |
| Индуизм | Без говядины, часто вегетарианское | — |
| Буддизм | Часто вегетарианское | — |
| Христианство | Пост (среда, пятница, Великий пост) | — |

**Религиозные праздники — влияние на поездки:**

```
Рамадан (Ислам):
├── Рестораны могут быть закрыты днём
├── Изменённый режим работы учреждений
└── Уважать пост при организации мероприятий

Шаббат (Иудаизм):
├── Не планировать вылеты в это время
├── Отели рядом с синагогой
└── Кошерные рестораны заранее

Дивали (Индуизм):
├── Высокий сезон в Индии
├── Повышенные цены
└── Раннее бронирование обязательно
```

**Специфические запросы:**

- **Молитвенная комната в отеле** — указать при бронировании
- **Коврик для молитв** — можно запросить в отеле
- **Направление на Мекку (Qibla)** — многие отели указывают в номерах
- **Кошерный ресторан** — найти ближайший к отелю заранее

### Medical

Медицинские требования — организация поездки с учётом состояния здоровья клиента.

**Медицинские услуги в авиа:**

| Код | Описание | Требования |
|-----|----------|------------|
| MEDA | Medical Case | Медицинское разрешение авиакомпании |
| OXYG | Oxygen | Кислородное оборудование |
| STCR | Stretcher | Носилки (выкуп нескольких мест) |
| POXY | Portable Oxygen | Портативный кислородный концентратор |

**Процедура для MEDA:**

1. Клиент получает медицинскую справку от врача
2. Заполняется форма MEDIF (Medical Information Form)
3. Отправляется в медицинский отдел авиакомпании
4. Ожидание одобрения (24-72 часа)
5. При одобрении — бронирование подтверждается

**Перевозка медикаментов:**

| Тип | Ручная кладь | Багаж | Документы |
|-----|--------------|-------|-----------|
| Таблетки | Да | Да | Рецепт желателен |
| Инъекции | Да (с справкой) | Нет | Справка обязательна |
| Наркотические | Да (ограниченно) | Нет | Рецепт + справка |
| Инсулин | Да | Нет (температура) | Справка от врача |

**Страхование:**

- Проверить покрытие существующих заболеваний (pre-existing conditions)
- Уточнить лимиты на эвакуацию и репатриацию
- Рекомендовать расширенное покрытие при хронических заболеваниях

**Координация с отелями:**

- Номер рядом с лифтом (минимум ходьбы)
- Мини-холодильник для медикаментов
- Контакты ближайшей больницы/аптеки

## Практическое применение

**Кейс 1: Пассажир-диабетик**

Клиент с диабетом летит Алматы — Лондон (7 часов).

**Чек-лист:**
- [ ] Заказать DBML (диабетическое питание) — SSR в PNR
- [ ] Рекомендовать взять инсулин в ручную кладь
- [ ] Подготовить справку от врача на английском
- [ ] Уточнить наличие холодильника для инсулина в отеле
- [ ] Предоставить контакты ближайшей больницы к отелю

**Кейс 2: Семья с ребёнком на коляске**

Семья с ребёнком 3 лет в детской коляске летит на отдых.

**Чек-лист:**
- [ ] Заказать CHML (детское меню)
- [ ] Оформить коляску как специальный багаж (gate check)
- [ ] Уточнить правила провоза детских вещей у авиакомпании
- [ ] Запросить детскую кроватку в отеле
- [ ] Предупредить о необходимости раннего прибытия с ребёнком

**Кейс 3: Религиозные требования корпоративной группы**

Группа из 10 человек (разные религии) на корпоративный семинар в Дубае.

**Чек-лист:**
- [ ] Собрать диетические предпочтения каждого участника
- [ ] При групповом меню: обеспечить халяль (база) + вегетарианские опции
- [ ] Учесть Рамадан при планировании дат (если актуально)
- [ ] Информировать участников об особенностях ОАЭ (дресс-код, алкоголь)
- [ ] Подготовить memo с культурными особенностями страны
