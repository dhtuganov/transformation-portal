После изучения этого материала вы сможете:

- Определять визовый режим для любого направления

### Карта безвизовых направлений

<Callout type="tip">
**Важно!** Безвизовый режим не означает полную свободу въезда. Всегда требуется паспорт со сроком действия 6+ месяцев, обратный билет и подтверждение финансовой состоятельности.
</Callout>

## Электронные визы (E-Visa)

### Что такое E-Visa?

Электронная виза — это **разрешение на въезд**, оформляемое онлайн без посещения консульства. Виза привязана к номеру паспорта и подтверждается электронным документом.

### Страны с E-Visa для граждан РК

| Страна | Стоимость | Срок получения | Срок пребывания |
|--------|-----------|----------------|-----------------|
| **Индия** | $25 | 3-5 дней | 30 дней |
| **Австралия** | $20 | 24-72 часа | 3 месяца |
| **Кения** | $51 | 2-3 дня | 90 дней |
| **Шри-Ланка** | $50 | 24 часа | 30 дней |
| **Мьянма** | $50 | 3 дня | 28 дней |
| **Камбоджа** | $36 | 3 дня | 30 дней |
| **Египет** | $25 | 5-7 дней | 30 дней |
| **Оман** | $20 | 24 часа | 10 дней |
| **Бахрейн** | $29 | 24 часа | 14 дней |
| **Пакистан** | $60 | 5-7 дней | 30 дней |
| **Турция** | Бесплатно | — | Безвизовый! |

### Документы для Шенгена

| Документ | Требования |
|----------|------------|
| **Анкета** | Заполненная, подписанная |
| **Паспорт** | Действителен 3+ мес. после выезда, 2 чистые страницы |
| **Фото** | 35x45 мм, белый фон, без улыбки |
| **Страховка** | Минимум €30,000, все страны Шенгена |
| **Бронь отеля** | Или приглашение от принимающей стороны |
| **Билеты** | Туда и обратно (бронь достаточно) |
| **Справка с работы** | На фирменном бланке, с зарплатой |
| **Выписка из банка** | Минимум €50-70/день поездки |
| **Подтверждение связи с РК** | Недвижимость, семья, бизнес |

## Timatic — официальный источник

### Что такое Timatic?

<Callout type="info">
**Timatic** — это база данных IATA с актуальной информацией о визовых, паспортных и медицинских требованиях. Используется всеми авиакомпаниями для проверки пассажиров перед посадкой.
</Callout>

### Как использовать Timatic

```
┌─────────────────────────────────────────────────────────────────────┐
│                    TIMATIC В AMADEUS                                │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  КОМАНДЫ:                                                           │
│  ────────                                                           │
│                                                                     │
│  TIMATIC/NA-KZ/DE-FR                                               │
│  └──────┬────────────┘                                             │
│         │                                                           │
│         └─ NA = Nationality (гражданство)                          │
│            DE = Destination (страна назначения)                    │
│                                                                     │
│  Пример результата:                                                 │
│  ─────────────────                                                  │
│  VISA REQUIRED                                                      │
│  Visa required, except for Holders of a valid Schengen             │
```

## Практические упражнения

### Упражнение 1: Определение визового режима

**Задание:** Клиент летит Алматы → Стамбул → Барселона → Алматы. Какие визы нужны?

<details>
<summary>Показать решение</summary>

**Анализ маршрута:**

1. **Алматы → Стамбул (Турция)**
   - Безвизовый режим для РК
   - 30 дней без визы
   - ✅ Виза не нужна

2. **Стамбул → Барселона (Испания)**
   - Испания = Шенгенская зона
   - Для граждан РК требуется виза
   - ⚠️ **Нужна Шенгенская виза**

3. **Барселона → Алматы**
   - Возвращение домой
   - ✅ Виза не нужна

**Итог:** Нужна **Шенгенская виза** в консульстве Испании.

</details>

### Упражнение 2: Расчёт сроков

**Задание:** Клиент хочет вылететь в Париж 15 марта. Сегодня 1 февраля. Успеет ли получить визу?

<details>
<summary>Показать решение</summary>

**Расчёт:**

- До вылета: 42 дня
- Срок рассмотрения Шенгена: 15-45 дней
- Сбор документов: 3-7 дней
- Запись в ВЦ: 1-5 дней

**Итого минимум:** 19 дней
**Итого максимум:** 57 дней

**Вывод:**
- Оптимистичный сценарий (19-25 дней) = ✅ Успеет
- Пессимистичный сценарий (45+ дней) = ❌ Риск не успеть

**Рекомендация:** Начать процесс НЕМЕДЛЕННО. Рассмотреть вариант другой страны Шенгена с более быстрым рассмотрением (Италия, Греция).

</details>

### Упражнение 3: Транзитные сложности

**Задание:** Клиент летит Алматы → Лондон (пересадка 5 часов) → Нью-Йорк. Какие визы нужны?

<details>
<summary>Показать решение</summary>

**Анализ:**

1. **Лондон (UK) — транзит 5 часов**
   - UK не входит в Шенген
   - Для транзита БЕЗ выхода из транзитной зоны: виза НЕ нужна
   - ✅ Если есть действующая виза США

2. **Нью-Йорк (США)**
   - Обязательна виза B1/B2
   - Интервью в посольстве
   - ⚠️ **Нужна виза США**

**ВАЖНО:** При наличии визы США транзит через UK без визы возможен. Без визы США — нужна транзитная виза UK!

**Рекомендация:** Сначала получить визу США, затем бронировать билет через Лондон.

</details>
