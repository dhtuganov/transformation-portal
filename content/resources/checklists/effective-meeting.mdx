# Чек-лист: Эффективное совещание

Эффективное совещание — это инвестиция времени с измеримым результатом. Используйте этот чек-лист для структурированной подготовки и проведения встреч.

## До совещания (подготовка)

### Определение необходимости

- [ ] Убедиться, что встреча действительно нужна (нельзя ли решить вопрос email/чатом?)
- [ ] Сформулировать четкую цель совещания
- [ ] Определить ожидаемый результат (решение, план, список задач)
- [ ] Понять, кто должен присутствовать обязательно, а кто опционально

### Организация

- [ ] Выбрать оптимальное время для всех участников
- [ ] Забронировать переговорную комнату / создать онлайн-встречу
- [ ] Отправить календарное приглашение с темой и целью
- [ ] Подготовить и разослать повестку дня заранее (минимум за день)
- [ ] Попросить участников подготовить материалы (если требуется)

### Подготовка материалов

- [ ] Собрать необходимые данные и документы
- [ ] Подготовить презентацию (если нужна)
- [ ] Проверить техническое оборудование заранее
- [ ] Подготовить шаблон для протокола/заметок
- [ ] Определить, кто будет вести протокол

> **Правило**: Совещание без повестки дня — это совещание без результата.

## Повестка дня (пример структуры)

Отправьте участникам заранее:

```
Тема: Планирование запуска продукта X
Дата: 15 декабря, 10:00-11:00
Место: Переговорная 3 / Zoom-ссылка

Цель: Согласовать план и сроки запуска

Повестка:
1. Текущий статус разработки (10 мин) - Иван
2. Маркетинговая стратегия (15 мин) - Мария
3. Необходимые ресурсы (10 мин) - Анна
4. Согласование сроков (15 мин) - все
5. Next steps и ответственные (10 мин)

Подготовить: данные по конкурентам (Мария)
```

## Во время совещания

### Начало (первые 5 минут)

- [ ] Начать вовремя (не ждать опоздавших)
- [ ] Поприветствовать участников
- [ ] Напомнить цель и повестку встречи
- [ ] Назначить таймкипера и протоколиста
- [ ] Установить правила (например, телефоны в сторону)

### Проведение

- [ ] Следовать повестке и тайминг-плану
- [ ] Давать слово всем участникам
- [ ] Фиксировать ключевые решения и action items
- [ ] Возвращать обсуждение в нужное русло при отклонениях
- [ ] Следить, чтобы все важные вопросы были рассмотрены

### Модерация дискуссии

- [ ] Не допускать перебиваний
- [ ] Поощрять конструктивные идеи
- [ ] Пресекать переход на личности
- [ ] Возвращать фокус на цель при затягивании
- [ ] Делать промежуточные выводы по каждому вопросу

### Завершение (последние 5-10 минут)

- [ ] Резюмировать принятые решения
- [ ] Зафиксировать список задач с ответственными и дедлайнами
- [ ] Убедиться, что все понимают свои next steps
- [ ] Назначить дату следующей встречи (если требуется)
- [ ] Поблагодарить участников за время

> **Важно**: Каждое решение должно иметь ответственного и срок выполнения.

## После совещания (в течение 24 часов)

### Документация

- [ ] Оформить протокол совещания
- [ ] Записать все принятые решения
- [ ] Составить список задач с ответственными и дедлайнами
- [ ] Разослать протокол всем участникам
- [ ] Сохранить протокол в общедоступное место

### Follow-up

- [ ] Создать задачи в таск-менеджере для всех action items
- [ ] Отправить напоминания ответственным
- [ ] Запланировать контрольные точки
- [ ] Поставить напоминание себе для проверки прогресса
- [ ] Собрать обратную связь (для важных встреч)

## Роли на совещании

### Модератор (ведущий)
- Следит за повесткой и временем
- Управляет дискуссией
- Резюмирует результаты

### Таймкипер
- Отслеживает время по каждому пункту
- Предупреждает о завершении времени
- Помогает не затягивать обсуждение

### Протоколист
- Фиксирует ключевые моменты
- Записывает решения и задачи
- Оформляет итоговый протокол

### Участники
- Готовятся заранее
- Активно участвуют в обсуждении
- Берут на себя ответственность за задачи

## Признаки неэффективного совещания

- ❌ Нет четкой повестки дня
- ❌ Начинается с опозданием
- ❌ Обсуждаются посторонние темы
- ❌ Нет конкретных решений и задач
- ❌ Участники не подготовились
- ❌ Затягивается за пределы времени
- ❌ Нет протокола после встречи
- ❌ Слишком много/мало участников

## Признаки эффективного совещания

- ✅ Четкая цель и повестка
- ✅ Начинается и заканчивается вовремя
- ✅ Все участники вовлечены
- ✅ Есть конкретные решения
- ✅ Задачи распределены с дедлайнами
- ✅ Протокол разослан в течение 24 часов
- ✅ Есть план дальнейших действий

## Советы по тайм-менеджменту

- **30-минутные встречи** вместо часовых (по возможности)
- **Парковка вопросов**: фиксируйте офтопик-темы для отдельного обсуждения
- **Правило 2 минут**: если вопрос решается за 2 минуты, решите его сразу
- **Буфер между встречами**: планируйте 5-10 минут между совещаниями
