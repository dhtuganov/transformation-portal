# MBTI Content Library Structure

## Overview

Created comprehensive content library for MBTI type profiles with detailed, gamified learning experiences.

## Structure

Each MBTI type has 7 dedicated articles organized in its own directory:

```
content/articles/
├── intj/
│   ├── overview.mdx              # Общий обзор типа
│   ├── cognitive-stack.mdx       # Когнитивные функции
│   ├── shadow-work.mdx           # Работа с тенью
│   ├── stress-patterns.mdx       # Стресс и выгорание
│   ├── growth-path.mdx           # Путь развития
│   ├── career-guide.mdx          # Карьера и профессия
│   └── relationships.mdx         # Отношения
└── enfp/
    ├── overview.mdx
    ├── cognitive-stack.mdx
    ├── shadow-work.mdx
    ├── stress-patterns.mdx
    ├── growth-path.mdx
    ├── career-guide.mdx
    └── relationships.mdx
```

- **Stress Patterns**: Триггеры, стадии стресса, стратегии восстановления
- **Growth Path**: 7 методов развития, годовой план, баланс

- **Overview**: Краткий портрет, энергия и харизма, статистика
- **Cognitive Stack**: Ne-Fi-Te-Si функции
- **Shadow Work**: Работа с ISTJ-тенью, развитие дисциплины
- **Stress Patterns**: Триггеры стресса, восстановление энергии
