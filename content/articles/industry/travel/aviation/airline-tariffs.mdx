## Введение в тарифообразование

<Callout type="info">
**Revenue Management (управление доходами)** — это наука о продаже нужного места нужному клиенту по нужной цене в нужное время. Авиакомпании используют сложные алгоритмы для максимизации дохода на каждый рейс.
</Callout>

## Классы бронирования (Fare Classes)

### Что такое класс бронирования?

Класс бронирования — это **буквенный код**, определяющий:
- Уровень цены
- Условия тарифа (возврат, обмен)
- Количество миль/баллов
- Доступность апгрейда

<Callout type="warning">
**Важно!** Класс бронирования ≠ класс обслуживания. Пассажир в классе Y и пассажир в классе Q летят в одном экономическом салоне, но платят разные цены и имеют разные условия.
</Callout>

## Правила тарифов (Fare Rules)

### Категории правил ATPCO

<Callout type="info">
**ATPCO (Airline Tariff Publishing Company)** — организация, которая стандартизирует и публикует тарифы авиакомпаний. Правила разделены на 35 категорий.
</Callout>

### Ключевые категории правил

```
┌─────────────────────────────────────────────────────────────────┐
│                  FARE RULES - ОСНОВНЫЕ КАТЕГОРИИ                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  CAT 4  - FLIGHT APPLICATION                                   │
│           На какие рейсы распространяется тариф                 │
│           (номера рейсов, дни недели)                           │
│                                                                 │
│  CAT 5  - ADVANCE RESERVATION/TICKETING                        │
│           Требования предварительного бронирования              │
│           Пример: "Бронировать за 21 день до вылета"           │
│                                                                 │
│  CAT 6  - MINIMUM STAY                                         │
│           Минимальный срок пребывания                          │
│           Пример: "Минимум 3 ночи" или "включая воскресенье"   │
│                                                                 │
│  CAT 7  - MAXIMUM STAY                                         │
│           Максимальный срок пребывания                         │
│           Пример: "Не более 30 дней"                           │
│                                                                 │
│  CAT 10 - COMBINABILITY                                        │
│           Возможность комбинирования с другими тарифами        │
│           (add-ons, круговые маршруты)                         │
│                                                                 │

│                                                                 │
```

### Как читать CAT 16 (Penalties)

**Пример из Amadeus:**

```
CATEGORY 16 - PENALTIES
────────────────────────────────────────────────────
CHANGES
  BEFORE DEPARTURE
    CHARGE USD 100.00 FOR REISSUE
    OR FARE DIFFERENCE WHICHEVER IS HIGHER
  AFTER DEPARTURE
    CHARGE USD 150.00 FOR REISSUE

CANCELLATIONS
  BEFORE DEPARTURE
    CHARGE USD 150.00 FOR REFUND
    TICKET IS NON-REFUNDABLE AFTER DEPARTURE

NO-SHOW
  CHARGE 100 PERCENT OF FARE FOR NO SHOW
────────────────────────────────────────────────────
```

**Расшифровка:**
- Обмен до вылета: $100 или разница в тарифе (что больше)
- Обмен после вылета: $150 + разница
- Отмена до вылета: штраф $150
- Отмена после вылета: возврат невозможен
- Неявка: потеря 100% стоимости

## Практические упражнения

### Упражнение 1: Чтение правил

**Задание:** Расшифруйте следующее правило CAT 16:

```
PENALTIES
BEFORE DEPARTURE
  CHARGE EUR 75.00 FOR REISSUE/REVALIDATION
  NOTE - WAIVED FOR DIFFERENT FARE COMP TO HIGHER FARE
AFTER DEPARTURE
  TICKET IS NON-REFUNDABLE AFTER DEPARTURE
  CHANGE FEE EUR 125.00 APPLIES
```

<details>
<summary>Показать решение</summary>

**Расшифровка:**
- **До вылета:**
  - Обмен стоит €75
  - Если переоформление на более дорогой тариф — штраф отменяется (только доплата разницы)
- **После вылета:**
  - Возврат невозможен
  - Обмен стоит €125

**Практический вывод:** Если клиент хочет изменить дату на более дорогую — берём только разницу. Если на более дешёвую — €75 + возможная доплата до нового тарифа.

</details>

### Упражнение 2: Расчёт стоимости

**Задание:** Рассчитайте полную стоимость:
- Базовый тариф: $380
- Топливный сбор YQ: $95
- Аэропортовые сборы: $67
- Багаж 23 кг: $45
- Выбор места: $18

<details>
<summary>Показать решение</summary>

```
  Базовый тариф:      $380.00
+ Топливный сбор YQ:  $ 95.00
+ Аэропортовые сборы: $ 67.00
+ Багаж 23 кг:        $ 45.00
+ Выбор места:        $ 18.00
────────────────────────────────
  ИТОГО:              $605.00
```

**Для клиента озвучиваем:** "$605 включая все сборы и багаж"

</details>

### Упражнение 3: Сравнение Branded Fares

**Задание:** Клиент летит Астана-Стамбул. Нужен багаж 23кг и гибкость обмена. Что выгоднее — EcoFly + добавки или ExtraFly?

| | EcoFly | ExtraFly |
|-|--------|----------|
| Тариф | $249 | $349 |
| Багаж 23кг | +$55 | Включён |
| Обмен | Нет | $30 |
| **Итого при обмене** | ? | ? |

<details>
<summary>Показать решение</summary>

**EcoFly:**
- Тариф: $249
- Багаж: $55
- Обмен: невозможен
- **Итого без обмена: $304**
- **При необходимости обмена: потеря билета**

**ExtraFly:**
- Тариф: $349
- Багаж: включён
- Обмен: $30
- **Итого без обмена: $349**
- **При обмене: $349 + $30 = $379**

**Вывод:** ExtraFly дороже на $45, но даёт защиту от потери всего билета. Для деловых путешественников — однозначно ExtraFly.

</details>

## Глоссарий

| Термин | Определение |
|--------|-------------|
| **ATPCO** | Airline Tariff Publishing Company — организация, публикующая тарифы |
| **Base Fare** | Базовый тариф без сборов и налогов |
| **Fare Class** | Буквенный код класса бронирования |
| **Fare Basis** | Полный код тарифа (класс + условия) |
| **YQ/YR** | Топливный/страховой сбор авиакомпании |
| **Tax** | Государственные налоги и сборы аэропортов |
| **Penalty** | Штраф за изменение или отмену |
| **Branded Fare** | Маркетинговый продукт с набором услуг |
| **Ancillary** | Дополнительная платная услуга |
| **Revenue Management** | Управление доходами |
| **TCO** | Total Cost of Ownership — полная стоимость владения |
| **OW/RT** | One Way / Round Trip — в одну сторону / туда-обратно |
