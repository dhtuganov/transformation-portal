## Введение

Бюджетирование — основа успешного мероприятия. Правильно составленный бюджет позволяет контролировать расходы, избегать неприятных сюрпризов и демонстрировать профессионализм заказчику. Перерасход бюджета — одна из главных причин недовольства клиентов.

Для сотрудников Otrar Travel навыки бюджетирования критичны: это основа для ценообразования, переговоров с поставщиками и управления ожиданиями клиента. Точный бюджет — залог прибыльного проекта.

## Ключевые понятия

### Cost categories

Категории расходов позволяют структурировать бюджет и контролировать затраты по направлениям.

**Основные категории расходов MICE:**

| Категория | Доля бюджета | Компоненты |
|-----------|--------------|------------|
| Размещение | 25-35% | Номера, upgrades, resort fee |
| Питание | 20-30% | Завтраки, обеды, ужины, кофе-брейки |
| Площадка | 10-20% | Аренда залов, оборудование |
| Логистика | 10-15% | Трансферы, перелёты, визы |
| Программа | 10-15% | Спикеры, активности, entertainment |
| Прочее | 5-10% | Подарки, материалы, страховка |

**Структура бюджета мероприятия:**

```
БЮДЖЕТ: Конференция "Лидеры 2024"
Участники: 50 человек | Дни: 3 | Локация: Боровое

РАЗМЕЩЕНИЕ                              $15,000
├── 25 DBL номеров × 2 ночи × $250      $12,500
├── 3 Suite × 2 ночи × $400             $2,400
└── Resort fee                          $100

ПИТАНИЕ                                 $12,000
├── Завтраки (включены в номер)         $0
├── Обеды × 3 дня × 50 чел × $30        $4,500
├── Кофе-брейки × 4 × 50 чел × $15      $3,000
├── Ужин День 1 × 50 × $40              $2,000
└── Гала-ужин × 50 × $50                $2,500

ПЛОЩАДКА                                $5,000
├── Конференц-зал (3 дня)               $3,000
├── 2 переговорных (3 дня)              $1,200
└── AV оборудование                     $800

ЛОГИСТИКА                               $6,000
├── Автобус Алматы-Боровое RT           $3,000
├── Мини-трансферы на площадке          $1,000
└── Микроавтобус для VIP                $2,000

ПРОГРАММА                               $8,000
├── Keynote спикер                      $3,000
├── Тимбилдинг программа                $3,500
├── Живая музыка (гала)                 $1,500

ПРОЧЕЕ                                  $4,000
├── Подарки участникам × 50 × $40       $2,000
├── Материалы (бейджи, папки)           $500
├── Фото/видео                          $1,000
├── Страховка                           $500

───────────────────────────────────────────
ИТОГО РАСХОДОВ                          $50,000
На человека                             $1,000
```

**Фиксированные vs переменные расходы:**

| Тип | Примеры | Особенности |
|-----|---------|-------------|
| Фиксированные | Аренда зала, спикер, AV | Не зависят от количества участников |
| Переменные | Питание, подарки, размещение | Прямая зависимость от численности |
| Полупеременные | Трансферы, координаторы | Ступенчатая зависимость |

### Contingency

Contingency (резерв) — запас бюджета на непредвиденные расходы.

**Стандартные размеры резерва:**

| Тип мероприятия | Резерв | Обоснование |
|-----------------|--------|-------------|
| Повторяющееся | 5-7% | Низкий уровень неопределённости |
| Новое направление | 10-15% | Незнакомые поставщики |
| Международное | 15-20% | Валютные риски, логистика |
| Сложная программа | 10-15% | Много переменных |

**На что тратится резерв:**

- Колебания валютного курса
- Дополнительное оборудование
- Непредвиденные запросы клиента
- Изменение количества участников
- Погодные условия (outdoor events)
- Форс-мажорные обстоятельства

**Управление резервом:**

```
Правила использования contingency:

1. Резерв закладывается, но НЕ включается в презентацию клиенту
2. Каждое использование — с обоснованием
3. Неизрасходованный резерв = дополнительная прибыль
4. Регулярная сверка: плановый vs фактический бюджет
```

**Пример расчёта с резервом:**

```
Базовый бюджет:           $50,000
Contingency (10%):        $5,000
────────────────────────────────
Внутренний бюджет:        $55,000
Margin (15%):             $8,250
────────────────────────────────
Цена клиенту:             $63,250
```

### ROI calculation

ROI (Return on Investment) — показатель возврата инвестиций в мероприятие.

**Формула ROI:**

```
ROI = (Выгода - Затраты) / Затраты × 100%
```

**Метрики для оценки ROI мероприятия:**

| Тип мероприятия | Метрики успеха |
|-----------------|----------------|
| Конференция продаж | Рост продаж, новые сделки |
| Тимбилдинг | Вовлечённость, retention |
| Запуск продукта | Освещение в СМИ, лиды |
| Инсентив | Достижение KPI участниками |

**Количественные показатели:**

- **Прямые продажи** — сделки, заключённые на/после мероприятия
- **Лиды** — количество новых контактов
- **Охват** — участники + онлайн-аудитория
- **NPS** — индекс лояльности участников

**Качественные показатели:**

- Удовлетворённость участников (анкетирование)
- Качество нетворкинга
- Знание бренда (до/после)
- Командный дух (для корпоративных)

**Пример расчёта ROI:**

```
МЕРОПРИЯТИЕ: Конференция дилеров
Затраты: $100,000
Результаты (6 месяцев после):
- Новые контракты: $500,000
- Рост продаж у участников: +15% ($300,000)

ROI = ($800,000 - $100,000) / $100,000 × 100% = 700%

Дополнительные выгоды (не монетизируемые):
- Укрепление лояльности партнёров
- Обратная связь по продуктам
- База для следующего мероприятия
```

## Практическое применение

**Кейс 1: Оптимизация бюджета без потери качества**

Клиент хочет конференцию на 100 человек, но бюджет ограничен $80,000 (должно быть ~$100,000).

**Стратегии оптимизации:**

| Статья | Было | Стало | Экономия |
|--------|------|-------|----------|
| Отель | 5* $200/ночь | 4*+ $150/ночь | $5,000 |
| Питание | Full service | Буфет | $3,000 |
| Кофе-брейки | 4 в день | 2 + напитки | $2,000 |
| Трансфер | Индивидуальный | Групповой | $4,000 |
| Подарки | $50/чел | $30/чел | $2,000 |
| Спикер | Международный | Локальный топ | $4,000 |
| **Итого** | | | **$20,000** |

**Что НЕ оптимизировать:**
- Качество AV-оборудования
- Питание (качество, не количество)
- Безопасность
- Ключевые элементы программы

**Кейс 2: Обоснование бюджета для клиента**

Клиент считает бюджет завышенным. Как обосновать?

**Структура презентации бюджета:**

1. **Сравнение с рынком**
   - «Средняя стоимость аналогичного мероприятия: $X»
   - «Наше предложение: $Y (на Z% ниже/выше)»

2. **Детализация**
   - Показать каждую строку
   - Объяснить выбор поставщиков

3. **Альтернативы**
   - Вариант A: Базовый ($X)
   - Вариант B: Стандарт ($Y)
   - Вариант C: Премиум ($Z)

4. **Ценность**
   - Что получает клиент
   - Потенциальный ROI

**Кейс 3: Контроль бюджета в процессе**

Мероприятие через 2 недели, текущий расход — 85% бюджета.

**Чек-лист контроля:**

```
АНАЛИЗ СИТУАЦИИ
├── Какие статьи перерасходованы?
├── Какие ещё не оплачены?
└── Остаток резерва?

ДЕЙСТВИЯ
├── Заморозить дополнительные расходы
├── Пересмотреть оставшиеся закупки
├── Предупредить клиента (если критично)
└── Искать компенсации (бартер, скидки)

ПРЕВЕНТИВНЫЕ МЕРЫ
├── Еженедельная сверка план/факт
├── Approval process для новых расходов
└── Фиксация цен в контрактах
```
