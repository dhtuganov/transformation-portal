## Введение

Планирование корпоративных мероприятий — комплексный процесс, требующий системного подхода. От качества планирования напрямую зависит успех события: соблюдение бюджета, удовлетворённость участников и достижение бизнес-целей заказчика.

Для сотрудников Otrar Travel владение методологией планирования мероприятий открывает доступ к высокомаржинальному сегменту MICE (Meetings, Incentives, Conferences, Exhibitions). Корпоративные клиенты ценят профессионализм и готовы платить за качественную организацию.

## Ключевые понятия

### Brief

Brief (бриф) — документ, содержащий ключевую информацию о мероприятии от заказчика.

**Структура брифа:**

| Раздел | Содержание | Пример |
|--------|------------|--------|
| Цели | Зачем проводится мероприятие | Повышение мотивации команды |
| Аудитория | Кто участники | 50 топ-менеджеров |
| Формат | Тип мероприятия | Выездная конференция |
| Даты | Когда, продолжительность | 15-17 марта, 3 дня |
| Бюджет | Общий / на человека | $50,000 / $1,000 pp |
| Локация | Предпочтения по месту | В пределах 3ч от Алматы |
| Особые требования | Специфические пожелания | VIP-спикер, гала-ужин |

**Чек-лист вопросов для брифинга:**

```
ЦЕЛИ И РЕЗУЛЬТАТЫ
├── Какова главная цель мероприятия?
├── Как измеряется успех?
└── Какое послевкусие должно остаться?

УЧАСТНИКИ
├── Сколько человек?
├── Кто они (должности, возраст)?
└── Есть ли VIP-персоны?

ЛОГИСТИКА
├── Гибкие или фиксированные даты?
├── Откуда приезжают участники?
└── Нужны ли трансферы?

КОНТЕНТ
├── Какая программа нужна?
├── Нужны ли спикеры/фасилитаторы?
└── Какие активности желательны?

БЮДЖЕТ
├── Общий бюджет утверждён?
├── Включён ли НДС?
└── Какие статьи приоритетны?
```

### RFP

RFP (Request for Proposal) — запрос предложения, отправляемый потенциальным поставщикам.

**Структура RFP для площадки/отеля:**

```
RFP #2024-001
Корпоративная конференция ТОО "АБВ"

ОБЩАЯ ИНФОРМАЦИЯ
├── Название: Ежегодная конференция продаж
├── Даты: 15-17 марта 2024
├── Участники: 50 человек
└── Контакт: Мария Иванова, +7 777 123 4567

ТРЕБОВАНИЯ К РАЗМЕЩЕНИЮ
├── 25 номеров DBL (2 ночи)
├── 5 номеров Suite для руководства
├── Ранний check-in: 10:00
└── Поздний check-out: 18:00

ТРЕБОВАНИЯ К ЗАЛАМ
├── Конференц-зал: 50 чел, театр
├── 2 переговорных: по 15 чел
├── Оборудование: проектор, экран, микрофоны
└── Wi-Fi: высокоскоростной

ПИТАНИЕ
├── День 1: обед, кофе-брейк (2), ужин
├── День 2: завтрак, обед, кофе-брейк (2), гала-ужин
├── День 3: завтрак, обед
└── Особые требования: 5 вегетарианцев, 2 халяль

ДОПОЛНИТЕЛЬНО
├── Тимбилдинг на 3 часа
├── Банкетный ужин с живой музыкой
└── Трансфер аэропорт-отель-аэропорт

СРОКИ
├── Дедлайн предложений: 1 февраля
├── Выбор финалиста: 5 февраля
└── Подписание контракта: 10 февраля
```

**Критерии оценки предложений:**

| Критерий | Вес | Оценка 1-5 |
|----------|-----|------------|
| Соответствие требованиям | 25% | |
| Цена | 25% | |
| Качество площадки | 20% | |
| Опыт и репутация | 15% | |
| Гибкость условий | 15% | |

### Site selection

Site selection — процесс выбора оптимальной площадки для мероприятия.

**Этапы выбора площадки:**

1. **Лонг-лист** (5-10 вариантов)
   - На основе брифа и требований
   - Географический охват
   - Предварительная оценка

2. **Шорт-лист** (2-3 варианта)
   - Отправка RFP
   - Анализ предложений
   - Предварительные переговоры

3. **Site visit** (инспекция)
   - Личный осмотр площадки
   - Встреча с менеджером
   - Проверка технических возможностей

4. **Финальный выбор**
   - Сравнительный анализ
   - Презентация клиенту
   - Согласование решения

**Чек-лист для site visit:**

```
ОБЩЕЕ ВПЕЧАТЛЕНИЕ
├── Первое впечатление при входе
├── Атмосфера и дизайн
└── Чистота и ухоженность

КОНФЕРЕНЦ-ЗАЛЫ
├── Размер и конфигурация
├── Освещение (естественное/искусственное)
├── Акустика
├── Климат-контроль
├── Техническое оснащение
└── Доступ для людей с ОВЗ

НОМЕРНОЙ ФОНД
├── Качество номеров
├── Вид из окна
├── Звукоизоляция
└── Wi-Fi скорость

ПИТАНИЕ
├── Ресторан (вместимость, меню)
├── Возможности кейтеринга
└── Качество еды (дегустация)

ДОПОЛНИТЕЛЬНО
├── Парковка
├── Зоны для нетворкинга
└── Возможности для тимбилдинга
```

### Contracts

Contract (контракт) — юридически обязывающее соглашение с поставщиком услуг.

**Ключевые разделы контракта на мероприятие:**

| Раздел | Содержание |
|--------|------------|
| Стороны | Заказчик и исполнитель |
| Предмет | Описание услуг |
| Сроки | Даты проведения, дедлайны |
| Стоимость | Детализация по статьям |
| Оплата | График платежей |
| Отмена | Условия и штрафы |
| Форс-мажор | Обстоятельства непреодолимой силы |
| Ответственность | Обязательства сторон |

**Условия отмены (типичные):**

```
Срок до мероприятия    Штраф
───────────────────────────────
> 60 дней              0%
30-60 дней             25%
14-30 дней             50%
7-14 дней              75%
< 7 дней               100%
```

**Важные пункты для переговоров:**

- **Attrition clause** — допустимое отклонение по количеству номеров (обычно 10-20%)
- **Cut-off date** — дата, до которой можно изменить количество
- **Complimentary rooms** — бесплатные номера (1 на 20-50 забронированных)
- **Meeting space** — условия предоставления залов
- **F&B minimums** — минимальные расходы на питание

## Практическое применение

**Кейс 1: Организация ежегодной конференции**

Клиент: Нефтегазовая компания, 120 участников, 2 дня, бюджет $120,000

**Таймлайн планирования:**

| Этап | Срок | Действия |
|------|------|----------|
| Брифинг | T-90 дней | Встреча, сбор требований |
| RFP | T-80 дней | Отправка в 5 отелей |
| Анализ | T-70 дней | Оценка предложений |
| Site visit | T-60 дней | Осмотр 2-3 финалистов |
| Контракт | T-50 дней | Переговоры, подписание |
| Детализация | T-30 дней | Программа, меню, логистика |
| Финальная сверка | T-7 дней | Подтверждение всех деталей |
| Проведение | T-день | Контроль исполнения |
| Отчёт | T+7 дней | Закрытие, feedback |

**Кейс 2: Срочный запрос на тимбилдинг**

Клиент просит организовать тимбилдинг на 30 человек через 2 недели.

**Действия:**
1. Экспресс-брифинг (телефон/zoom)
2. Проверка availability 2-3 проверенных площадок
3. Отправка предложения в течение 24ч
4. Быстрое согласование (1-2 дня)
5. Контракт в упрощённой форме
6. Организация за 7-10 дней

**Риски короткого срока:**
- Ограниченный выбор площадок
- Невозможность переговоров по цене
- Меньше времени на подготовку контента
